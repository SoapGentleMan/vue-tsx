# admin

## 安装依赖
```
npm install
```

## 启动项目
```
npm start
```

## 打包项目
```
npm run release
```

## 启动mock服务
```
npm run mock-server
```

## 修改代码内跳转的域名指向
```
修改webpack DefinePlugin 的 CLIENT
```

## 修改服务端域名指向
```
修改webpack DefinePlugin 的 SERVER
```
